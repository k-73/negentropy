# Negentropy

## Quick Start

```bash
# Build
mkdir cmake-build-debug && cd cmake-build-debug
cmake ..
make

# Run
./negentropy
```

## Dependencies

- SDL2 (graphics/input)
- ImGui (UI framework)
- pugixml (XML serialization)
- GLM (math library)
- spdlog (logging)
- magic_enum (enum reflection)
- EnTT (entity component system)
